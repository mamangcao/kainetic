# Kainetic

**Kainetic** is a premium, high-performance fitness analytics dashboard built with **React**, **Tailwind CSS**, and **Framer Motion**.  
Designed as a third-party companion for Strava, it visualizes athlete data (running, cycling, walking) with a focus on aesthetics and user motivation.  

> **Kainetic is an independent dashboard and is not affiliated with, endorsed by, or sponsored by Strava, Inc.**

---

## Key Features

- **Deep Dark Mode UI**: "Deep Obsidian" theme with neon accents (Lime/Blue) and glassmorphism effects.  
- **Interactive Visualizations**: Dynamic Area Charts (via Recharts) and GitHub-style Consistency Heatmaps.  
- **Gamified Stats**: Displays "Best Efforts" (PRs) and custom Trophy achievements.  
- **Smooth UX**: Framer Motion for entrance animations, hover states, and seamless tab switching.  
- **Dual Access**: Supports simulated Strava OAuth login and a fully functional Demo Mode for guest users.  
- **Privacy-Focused**: Client-side architecture prioritizing data privacy and compliance with Strava's API terms.
